# 🪒 تطبيق مركز المليونير للحلاقة

تطبيق حجز مواعيد احترافي لصالونات الحلاقة الراقية باستخدام Flutter مع Clean Architecture والتصميم المتجاوب.

## 🎨 الألوان المخصصة

- **الأسود (#000000)**: النصوص الرئيسية  
- **الأبيض (#FFFFFF)**: الخلفيات والكروت
- **الأحمر الداكن (#A62424)**: الأزرار والإبرازات
- **الذهبي (#B6862C)**: العناصر المميزة والVIP

## ✨ الميزات

- نظام حجز المواعيد الذكي
- إدارة الخدمات والأسعار
- نظام الولاء والنقاط
- العروض والخصومات
- تقييم الخدمات
- إشعارات VIP
- تصميم متجاوب

## 🚀 بدء التطوير

1. **تثبيت المكتبات**
```bash
flutter pub get
```

2. **تشغيل التطبيق**
```bash
flutter run
```

## 🏗️ البنية المعمارية

يستخدم التطبيق Clean Architecture مع:
- **Presentation Layer**: Flutter BLoC
- **Domain Layer**: Use Cases & Entities  
- **Data Layer**: Repositories & Data Sources
